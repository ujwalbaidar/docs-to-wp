Congratulation! Your account has been created. Now connect Docs To WP to your WordPress website and start exporting. It will take less than 5 minutes.
Enter Your Website Domain (URL): This Field is Required! This means the URL where you generally log in to your WordPress Admin Panel but without the /wp-admin part. So if you generally log into https://docstowp.com/wp-admin, only put http://docstowp.com. Username: This Field is Required! Enter your WordPress website username i.e. what you use to login at https://yourwebsite.com/wp-admin. Make sure you have "Author" level permissions or above (otherwise Docs to WP won't work). Password: This Field is Required! Enter the password for the above username.
